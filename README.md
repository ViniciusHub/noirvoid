# NOIRVOID — Streetwear Concept Store

Página estática para GitHub Pages com visual de loja streetwear sofisticada.

## Como usar

1. Troque o nome `NOIRVOID` pelo nome da sua marca.
2. Troque o WhatsApp no arquivo `index.html`.
3. Troque `seuemail@email.com` no form por seu e-mail ou remova o formulário.
4. Troque o Instagram no footer.
5. Suba os arquivos no GitHub.
6. Ative GitHub Pages em Settings > Pages > Deploy from a branch > main > root.

## Arquivos

- `index.html`: estrutura da página.
- `style.css`: visual da loja.
- `script.js`: menu mobile, header e animações.
